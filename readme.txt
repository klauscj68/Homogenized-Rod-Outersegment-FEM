This directory contains the codes needed to run the homogenized finite element code. 
A user first defines the model parameters in the common/data.m file.
Then the user runs id/main_id.m
Then the user runs cyto/main_cyto.m

Volumic data will be recorded by the .mat files generated in cyto/
Data for surface, G-protein activation will be recorded by the .mat files generated in id/
